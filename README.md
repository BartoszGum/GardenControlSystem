# GardenControlSystem

-> Master:
 -Blynk connection
 -Turn on/off lights 
 -User communication
 -Motion sensor (PIR)
 -Light sensor 
 -Buttons
  
-> Communicator:
 -Reading sensors data and giving it to Master using I2C
 -Has create own WiFi to communication with wireless sensors
  
-> Sensor - software for wireless sensor (not ready jet)
